# Gremia.SBV 0.4.19 – Fallakte kompakter, Aktionen als Toast

## Ziel
Die Fallakte soll mehr vertikalen Arbeitsraum für die Bearbeitung bieten.

## Änderungen
- große Überschriften in der Fallseite reduziert
- Fallregister-Panel verdichtet
- Fallliste in scrollbare Tabellenhülle mit sticky Header verlegt
- permanente Rückmeldungen im unteren Bereich entfernt
- neue temporäre Toast-Rückmeldungen oben im Sichtbereich

## UX-Regeln
- Rückmeldungen erscheinen 4,2 Sekunden als Overlay
- Register bleibt kompakt
- Detailbereich behält Priorität gegenüber Listenhöhe
