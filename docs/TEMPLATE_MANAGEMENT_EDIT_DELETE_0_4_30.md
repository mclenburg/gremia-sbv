# Gremia.SBV 0.4.30 – Vorlagen bearbeiten und löschen

## Änderungen
- Die Vorlagen-Seite erzeugt keine Entwürfe mehr.
- Bestehende Vorlagen können bearbeitet werden.
- Bearbeitet werden können:
  - Titel
  - Kategorie
  - Maßnahmenstatus
  - Normen
  - Tags
  - Beschreibung
  - Betreff
  - Text
- Löschen erfolgt über einen kleinen Papierkorb-Button rechts in der Vorlagenübersicht.

## Hinweis
Die konkrete Nutzung der Vorlage bleibt in Fallakte oder Maßnahme.
