# Gremia.SBV 0.8.3 – Tätigkeitsbericht / anonymisiertes Reporting

## Ziel

0.8.3 ergänzt einen anonymisierten SBV-Tätigkeitsbericht im Modul Berichte.

## Inhalt

Der Bericht enthält:

- Fallakten gesamt
- Kontakte gesamt
- offene / überfällige Fristen
- Präventionsverfahren
- BEM-Verfahren
- Gleichstellung-/GdB-Verfahren
- Kündigungsanhörungen
- Statusauswertungen je Fachmodul
- Prüfliste vor Weitergabe

## Datenschutz

Der Bericht enthält bewusst keine:

- Namen
- Aktenzeichen
- Diagnosen
- Gesprächsnotizen
- Arbeitgebervorträge
- SBV-Stellungnahmen
- sonstigen sensiblen Freitexte

## Export

- Markdown-Export
- PDF-Export über druckoptimierte PDF-Ansicht
