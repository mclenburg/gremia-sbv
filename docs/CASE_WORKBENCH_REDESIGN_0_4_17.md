# Fallakten-Workbench 0.4.17

Die Fallakte ist das führende Arbeitsinstrument. Der Baum enthält keine Aktionen zum Anlegen neuer Vorgänge mehr, sondern ausschließlich Aktenbestandteile.

## Aufbau

- Fallliste oben
- Fallbaum links
- Bearbeitungsbereich rechts
- Footer-Aktionsleiste für neue Einträge

## Footer-Aktionen

- Notiz / Protokoll
- Dokument
- Frist
- Präventionsverfahren
- BEM
- Kündigungsanhörung
- Gleichstellung

## Bearbeitung

Ausgewählte Aktenbestandteile werden rechts bearbeitet. Präventionsverfahren können dort fortgeschrieben werden. Noch nicht vollständig implementierte Fachmodule werden weiterhin als fallbezogene Vormerkung angelegt.
