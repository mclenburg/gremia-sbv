# Fristen und Wiedervorlagen

## Zweck

Fristen und Wiedervorlagen schützen davor, dass wichtige Schritte untergehen. Gerade SBV-Arbeit ist häufig zeitkritisch: Beteiligungen, Anhörungen, Nachfragen, BEM-Schritte, Präventionsverfahren und Arbeitgeberreaktionen müssen nachgehalten werden.

## Frist oder Erinnerung?

Eine Frist hat rechtliche oder fachliche Bedeutung. Eine Wiedervorlage ist eine organisatorische Erinnerung.

Beispiele für Fristen:

- Stellungnahmefrist,
- Nachholung einer Beteiligung,
- Rückmeldefrist für Unterlagen,
- Termin zur erneuten Prüfung.

Beispiele für Wiedervorlagen:

- bei Beschäftigter Person nachfragen,
- Arbeitgeberantwort prüfen,
- Unterlagen sortieren,
- Gespräch vorbereiten.

## Gute Fristbeschreibung

Eine Frist sollte später ohne Kontextverlust verständlich sein. Notiere:

- worum es geht,
- was zu prüfen ist,
- wer handeln muss,
- welcher Vorgang betroffen ist,
- was passiert, wenn nichts geschieht.

## Abschluss einer Frist

Beim Abschließen sollte klar sein:

- was erledigt wurde,
- ob ein Ergebnis vorliegt,
- ob eine Folgefrist nötig ist,
- ob ein Journaleintrag sinnvoll ist.

Gremia.SBV kann aus bestimmten abgeschlossenen Fristen einen Journalvorschlag erzeugen. Dieser Vorschlag wird erst gespeichert, wenn du ihn bewusst bestätigst.

## Priorität

Nicht jede Frist ist gleich kritisch. Verwende Prioritäten, damit dringende Vorgänge sichtbar bleiben. Eine Frist mit Beteiligungs- oder Kündigungsbezug sollte besonders sorgfältig behandelt werden.

## Datenschutz

Fristen sollten keine unnötigen Diagnosen oder privaten Details enthalten. Der Titel darf knapp sein. Details gehören in die Fallakte oder Maßnahme, wenn sie erforderlich sind.


## Stellenbesetzungen

Stellenbesetzungsverfahren können Wiedervorlagen auslösen, etwa zur Nachforderung fehlender Unterlagen, zur Anhörung der SBV vor der Auswahlentscheidung oder zur Nachhaltung einer Arbeitgeberentscheidung. Diese Fristen können fallaktenunabhängig mit dem Stellenbesetzungsvorgang verknüpft werden.
