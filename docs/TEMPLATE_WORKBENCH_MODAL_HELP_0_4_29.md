# Gremia.SBV 0.4.29 – Vorlagen-Workbench

## Änderungen
- Neue Vorlage wird nicht mehr dauerhaft als Formular auf der Seite angezeigt.
- Oben im Vorlagenkatalog gibt es einen Button „Neue Vorlage“.
- Das Formular zum Anlegen einer Vorlage öffnet in einem Overlay-Modal.
- Oben links im Vorlagenkatalog gibt es ein Hilfe-Icon.
- Das Hilfe-Overlay erklärt die verfügbaren Platzhalter und deren Verwendung.
- Label, Eingabefelder und Filter wurden entzerrt.

## Ziel
Die Vorlagen-Seite bleibt Übersicht und Verwaltung. Konkrete Eingaben erfolgen fokussiert im Modal.
